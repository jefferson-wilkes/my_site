## Project

A personal profile website for hosting a simple game, built with React + Vite and styled with Tailwind CSS.

## Tech Stack

- **React + Vite** — frontend framework and build tool
- **Tailwind CSS** — styling

## Commands

- `npm run dev` — start local dev server
- `npm run build` — production build
- `npm run preview` — preview production build locally
- `npm run lint` — run ESLint

## Rules

- Keep the site simple and fast — avoid unnecessary dependencies
- Use semantic HTML for accessibility
- All layouts must be responsive (mobile-first)
